# shadcn/ui components

This folder is reserved for shadcn/ui components. To scaffold them, run:

```bash
npx shadcn-ui@latest init
npx shadcn-ui@latest add button card dialog dropdown-menu input label table select tabs badge calendar toast
```

Ensure Tailwind uses the custom palette defined in `tailwind.config.js`. Generated components can live here (or adjust the config to your preferred path).
